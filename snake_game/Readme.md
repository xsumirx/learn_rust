## A basic snake game implementation in Rust.

>### Features:
- [x] Snake walks, eats and grow
- [ ] Scoreboard 
- [ ] Collision Detection
- [ ] Start Menu 
- [ ] Sound


>### Keyboard control
```
Space           - Pause the game
Arrow Left      - Snake Move Left
Arrow Right     - Snake Move Right
Arrow Down      - Snake Move Down
Arrow Up        - Snake Move Up
```
